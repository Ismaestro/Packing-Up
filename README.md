# Packing Up App #

### What is this repository for? ###

* Build an app to make a list of things you have to carry on in any trip

### How do I get set up? ###

Ionic 2 app. Just run:

`npm install`

`npm start`

### Who do I talk to? ###

* Repo owner or admin
